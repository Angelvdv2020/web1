# noryx-web1

Express-сервер, который отдает локализованные HTML-страницы и локальные ассеты для сайта.

## Требования

- Node.js 18+
- npm

## Быстрый старт

```bash
npm install
npm run start
```

Сервер запускается на `PORT` из окружения, по умолчанию на `3002`.

## Переменные окружения

- `PORT` — порт HTTP-сервера (по умолчанию `3002`)
- `STATIC_MAX_AGE_MS` — TTL кэша для статических ассетов в миллисекундах (по умолчанию `604800000`, 7 дней)

## Проверка работоспособности

- `GET /healthz` → `200` и JSON `{ "ok": true }`

## Маршрутизация

- HTML-маршруты отдаются через `pageRoutes` с on-the-fly локализацией.
- Статические файлы обслуживаются из `public/`.
- Ассеты `/карта мира_files` берутся с приоритетом из `123/карта мира_files`, fallback — `public/карта мира_files`.

## Продакшн-рекомендации

- Запускать через process manager (`systemd`/`pm2`).
- Публиковать через reverse proxy (`Nginx`/`Caddy`) с TLS.
- Секреты хранить в vault/CI secrets, не в `.env` в репозитории.
