What is an IP address?

IP address is an unique identifier of the device on the internet or any local network. An IP stands for "Internet protocol", it's basically a string of characters that helps devices "talk" to each other online. Simply put, an IP address is a number that links all your online activity to your computer, mobile phone, or any other device. Whenever you open a specific website, watch movies, use any apps, they get your IP address and send content back to you. An IP address can be used to track you down as well because it belongs to your ISP (Internet Service Provider). By using a VPN, you can change your IP address and nobody will be able to find out where the traffic really comes from.

IPv4 vs. IPv6 addresses

IPv4 is a version of IP (Internet Protocol) that is most widely used nowadays. IPv4 is the identifier of devices on a network using a 32-bit address scheme. Today, IPv4 is the most commonly used version of IP. Basically, IPv4 is a string written in four numbers separated by dots. In most cases, this address is unique for every device. For example, here is how IPv4 looks like: 95.141.37.24. As the world population grows, the number of unique IPv4 IP addresses simply ends. IPv4 produces only 4 billion addresses and world population has exceeded this number a long time ago. That's why, the new generation of IP saw the world - IPv6. The IPv6 is a 128-bit hexadecimal address. For example, here is an IP address in the IPv6 format: 2001:0db8:85a3:0000:0000:8a2e:0370:7334. In simple words, IPv6 is longer and is capable of further growing. Yet, IPv6 is not as popular as IPv4, but we will certainly see the growth of its usage soon. One more difference is that IPv6 is an alphanumeric address while IPv4 is only numeric one. There are other key differences including security features, end-to-end connection integrity, packet flow identification, and more.

What is a public IP address?

A public IP address is an IP address that identifies your device on the Internet. Your ISP (Internet Service Provider) assigns an IP address to your Wi-Fi router or device via the Ethernet cable. This IP address is publicly visible and used to establish connection over the Internet. Other devices around the world use this IP to identify your device, transfer content and other information to you. Given that your public IP is visible to everyone, it's important to keep it safe and private. It's where a VPN comes in handy because it hides your real IP address.

What is a private IP address?

A private IP address is an IP address of your device that can be used in local networks only. Your private IP address is hidden when you are browsing the Internet. The traffic can't be sent from private IP address to the Internet. Basically, the main difference between private and public IP address is that private IP address can be used only within local or LAN networks in the office, enterprise, or any other community. When you go online, nobody will see your private IP address, but your public IP address assigned by your Internet Service Provider will be visible.

Public vs. private IP addresses

The main difference betweeen public and private IP addresses is the place where they are used. Your public IP address is visible to the whole Internet and every website you visit. This IP address is the identifier of your device in the public network. Private IP address is not visible to everyone - it identifies your device only within your private or LAN network.

Static vs. dynamic IP address

Think of an IP address as a way to distinguish one device connected to the internet from another – a digital alternative for a physical address or a social security number. There are two types of IP addresses: static and dynamic. Despite what you might think, the difference is in the name and is quite simple to remember. The static IP address is fixed and stays the same, while the dynamic IP address changes over time and can be assigned to another host by DHCP, a network server for automated assignment of unutilized IP addresses to a pool of available ones.

Static public IP addresses

Static public IP addresses are permanently assigned by the Internet Service Providers. The static IP address is more precise and can give more accurate information about the computer with said IP address, such as physical location. Most businesses utilize the static IP address, making it easier for their customers to find them via the Domain Name Service. This type of IP address is also necessary to have a reliable connection, run a web server or e-mail server, manage the remote networking services, and properly operate computers, peripherals, and other devices with each other by reaching a specific server. An obvious downside of a static IP address is the possibility of tracing the physical address of a computer assigned to the IP address in case of a security violation. Besides that, static IP addresses cost more, requiring additional manual support instead of automated DHCP server assignment.

There are public static IP addresses and the private ones, the first being fixed, visible to the world, and provided by the ISP. In contrast, private static IP addresses are fixed but not visible to the internet and assigned manually.

Dynamic public IP addresses

Dynamic IP addresses are assigned and constantly changed by the DHCP server of the Internet Service Provider with no manual interference on the user’s side. They are usually easy to manage and allow reusing multiple IP addresses in the network.

The two main advantages of dynamic IP addresses are lower costs compared to static ones and the lower possibility of being tracked or hacked due to constant changes in the IP address. While web servers or companies use static IP addresses, dynamic IP addresses are more user-oriented. However, the constant change of the dynamic IP address makes it harder to whitelist multiple devices and operate with remote access.

Dynamic IP addresses are preferred by average consumers and aren’t really suitable for businesses’ needs. This type of IP address is suitable for basically all smart devices such as laptops, PCs, tablets, and mobile phones. Mobile networks such as 3G or 4G generally have dynamic IP addresses, however, some providers may offer a static IP as an option for an additional monthly fee.

The IP address may become a gateway to your personal information if it falls into the wrong hands. It is not that hard to find out where you live or your name just by knowing your IP address, especially if it is static. There are multiple ways the hackers can get a hold of your IP address – using a torrent to download and seed content, clicking on shady links or advertisements on the internet can do the trick.

So what can someone do with your IP address?

Let’s just start with the least threatening category in this list – ISPs. They can track your internet activity without you ever knowing about this kind of surveillance, while some unreliable Internet service providers have a history of selling their users’ personal information to advertisers for financial gain, and they don’t even need to fool you into clicking on some alluring advertisement to reveal your IP address. Hackers can do you worse by tracking you online and revealing your physical address, hacking into your PC or mobile phone, infecting your device with malware, or even stealing your identity.

Frequently Asked Questions

Who can use my IP address?

Your IP address can be used by various hackers and snoopers if they get access to your network. If you install malware or suspicious apps on your device, your IP can also be compromised. It’s important to avoid visiting suspicious websites and installing applications you don’t trust. If you don’t want your IP address to be compromised, it’s recommended to use a VPN on a daily basis.

Does your IP address tell where you live?

Your IP address can tell where you live approximately. Every IP address is bounded to a specific location. Although this location usually doesn’t show the street and the building where you are located, it’s possible to track you down. If you have a signed contract with your Internet Service Provider (ISP), it won’t take a lot of time for government agencies to get your physical address.

Can an IP address identify a person?

An IP address can help identify a person if necessary. All IP addresses are assigned by Internet Service Providers (ISPs). Every person has a signed contract with ISP where customers usually leave their personal information. ISPs know exactly which IP addresses are used by their customers. That is why, it is certainly possible to identify a person knowing their public IP address.

How to secure my IP address?

The best way to secure your IP address is to use a reliable VPN service. A VPN will hide your IP address and it won’y be shown publicly. Once you connect to a VPN, you will get another IP address that can’t be tracked down to you. This way, you secure your real IP address.

Do I need to check my IP address?

You probably need to check your IP address a few times a month. If you share your IP address with someone, make sure it’s not blacklisted. If your IP address becomes suspicious it might cause some problems. If you want to check your IP address, open this page.

Why is my IP address location wrong?

There are many reasons why your IP address location can be wrong. First of all, all databases with IP addresses and their locations are different. There are different sources of IP location that may show different results. Some databases update too seldom to display accurate data. Another reason is the fact that ISPs can assign new IP addresses and remove old ones. This way, databases of IP location data can take too long to update, so the information won’t be accurate.
